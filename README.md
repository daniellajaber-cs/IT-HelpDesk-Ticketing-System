# IT-HelpDesk-Ticketing-System
A full stack web-based IT Help Desk and Ticketing Management System for managing support requests, users, tickets, reports, and workflows.
## Selected Technology Stack
- Frontend: React.js
- Backend: ASP.NET Core Web API
- Database: SQL Server + SSMS
- Version Control: Git & GitHub
- API Testing: Postman
