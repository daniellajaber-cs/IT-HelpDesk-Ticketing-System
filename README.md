# IT Help Desk & Ticketing Management System

## Project Overview

The IT Help Desk & Ticketing Management System is a web-based application designed to help organizations manage technical support requests efficiently. Employees can submit support tickets, while IT Support Agents and Administrators can track, assign, update, and resolve issues through a centralized platform.

The system improves communication between employees and support teams while providing an organized workflow for ticket management.

---
## Selected Stack 
*Frontend: React.js
*Backend:ASP.NET Core Web API
*Database:SQL Server + SSMS 
*Version Control:Git & GItHub
API Testing: Postman
